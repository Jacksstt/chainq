# @chainq/ingest-evm

EVM data ingestion. Backfill via [cryo](https://github.com/paradigmxyz/cryo), realtime via [Subsquid](https://github.com/subsquid/squid-sdk).

## Prerequisites

```bash
# cryo (Rust binary)
cargo install cryo_cli
# or
brew install paradigmxyz/brew/cryo
```

## Status

Pre-alpha. Real cryo wiring lands in `v0.0.1`.
