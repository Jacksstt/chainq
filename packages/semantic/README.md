# Semantic Layer

YAML metric definitions that an AI agent can read directly. Each metric maps a name
(e.g. `dex_volume_usd`) to a SQL template, dimensions, filters, and guardrails.

Schema is documented in [`../../docs/ARCHITECTURE.md`](../../docs/ARCHITECTURE.md#l4--semantic-layer).

Example metrics live under `metrics/` once we start filling them in (v0.0.1).
